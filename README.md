# 大模型行业经营数据分析工具

一个简单易用的 Web 应用，用于分析大模型行业的经营数据。

## 功能特点

- 📁 支持拖拽或点击上传 Excel 文件
- 💾 自动将数据保存到 SQLite 数据库
- 📊 自动计算环比增长率（MoM）
- 📋 实时预览分析结果
- 📥 一键下载 Excel 报告
- 📜 历史月份数据管理

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python3 run.py
```

### 3. 打开浏览器

访问: http://localhost:5000

## 使用说明

1. **上传 Excel 文件**: 点击或拖拽文件到上传区域
2. **选择月份**: 输入月份（格式: YYYY.MM，例如 2026.5）
3. **保存数据**: 点击"保存数据"按钮将数据存入数据库
4. **生成报告**: 点击"生成报告"按钮计算环比增长率
5. **下载报告**: 点击"下载 Excel 报告"按钮保存结果

## Excel 文件格式要求

必须包含以下字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 时间 | 月份 | 2026.5 |
| 模态 | 产品类型 | 文本生成 |
| 模型 | 模型名称 | HY 2.0 Think |
| Token | Token 数值 | 108.4 |
| 收入 | 收入数值 | 119.6 |
